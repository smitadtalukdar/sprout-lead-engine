# 🌱 Sprout Lead Engine

Automated lead generation and outreach system for Sprout Growth Agency.

## What it does
- Discovers SMB leads across India and neighbouring countries (one industry + one region at a time)
- Runs a brand audit for each lead
- Generates a personalised growth report
- Sends max 3 outreach emails/day via Brevo
- Sends personalised follow-ups after 7 days

## Stack
- **AI:** Groq (Llama 3.3 70B) — free
- **Email:** Brevo SMTP API — free
- **Hosting:** GitHub Pages — free
- **Storage:** Browser localStorage — no database needed

## Setup
1. Open the live URL
2. Click ⚙️ Setup Keys
3. Add your Groq API key (console.groq.com)
4. Add your Brevo API key (app.brevo.com)
5. Start discovering leads
